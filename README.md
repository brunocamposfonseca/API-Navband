# API de Consulta de Localização

## Descrição

Esta API tem o propósito de interligar sistemas (site, aplicação mobile e hardware) permitindo a consulta e criação de: usuários, localização e pulseiras, permitindo a integração com websites, dispositivos móveis e sistemas de hardware. Ela permite buscar informações de localização com base em consultas específicas, oferecendo respostas rápidas e precisas.

## Uso e Restrições

- Esta API é disponibilizada apenas para fins de consulta e visualização.
- O código-fonte e a infraestrutura da API são de propriedade exclusiva de NavBand.
- Qualquer uso comercial, modificação ou distribuição sem autorização prévia é estritamente proibido.

## Requisitos

- A API pode ser utilizada em sites, dispositivos móveis ou em hardware com integração customizada.
- Consulte a documentação abaixo para saber os endpoints e formatos de retorno.
